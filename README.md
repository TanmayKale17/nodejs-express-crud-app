A basic CRUD application using Node.js, Express, and MongoDB. 

## Features
- RESTful API with CRUD operations
- MongoDB with Mongoose
- MVC structure

## Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose

## How to Run
1. Clone the repo
2. Run `npm install`
3. Set up `.env` with MongoDB URI
4. Run `npm start` or `nodemon server.js`
